# 🎮 RE4 Endless Runner – Pixel Mini Game

A fast-paced 2D endless runner inspired by **Resident Evil 4**, built using vanilla JavaScript and HTML5 Canvas.

This project combines classic runner mechanics with a stylized pixel-art horror theme, featuring animated characters, seamless ground movement, and custom collision systems.

---

## 📚 About The Project

This is a browser-based endless runner where the player controls a character (Leon-inspired) who must jump over incoming obstacles (Ashley 👀).

The game focuses on **game feel, animation, and rendering systems**, rather than using any external libraries or engines.

Players can:

- Jump to avoid obstacles
- Survive as long as possible to increase score
- Experience animated sprites and scrolling environments
- Restart the game after losing

---

## 🧠 Key Concepts

This project explores core game development fundamentals:

- Game loop architecture using `requestAnimationFrame`
- Physics simulation (gravity, velocity, jump mechanics)
- Collision detection using custom hitboxes
- Separation of visual sprites and logical hitboxes
- Sprite-based animation systems
- Infinite world illusion through object recycling
- Event handling for input and UI (keyboard + mouse)
